# 任务拆解机

一个静态网页工具，用来把模糊的大任务拆解成更具体、可执行的微步骤。

## 功能

- 支持填写并保存 API Key 等内容
- 可将任务拆解为多个微步骤卡片
- 点击卡片即可移除已完成步骤
- 支持重新生成新的拆解结果
- 纯前端实现，无需构建工具

## 本地使用

1. 克隆或下载项目到本地
2. 直接用浏览器打开 `index.html`
3. 填写 API 配置后即可开始使用

## 使用说明

1. 点击右上角 `设置`，打开 API 配置侧边栏
2. 选择供应商，填写 API Key，并确认模型
3. 点击 `拉取模型` 或 `测试连通性`
4. 在页面中输入你想拆解的任务
5. 点击 `拆解任务`，生成微步骤卡片
6. 点击某张卡片，可将该步骤从页面中移除
7. 点击 `重新生成`，可基于当前任务重新生成新的卡片结果

## 注意事项

- 本项目是纯前端页面，API Key 保存在当前浏览器的 IndexedDB 中
- 清除浏览器站点数据后，已保存的配置和卡片内容会丢失
- 更换设备或浏览器后，本地保存内容不会自动同步
- 如果直接本地打开页面，某些浏览器可能会缓存旧文件，遇到异常时可先强制刷新
- 发布到 GitHub Pages 时，直接使用仓库根目录即可，无需构建步骤
